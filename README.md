## Go Missing Utilities

### Overview
This project provides some useful utilities that are missing from the standard
Go libraries.  It provides functions that help with the following areas:
 * Strings
 * Arrays and Maps
 * Filesystem
 * Random Numbers
 * Encoding/Decoding
 * Reflection
